## üõ†Ô∏è How to Use

### ‚úÖ Requirements
- Python 3.6 or above
- Internet connection

### üîß Steps

1. **Download or Clone this Repository**
   ```bash
   git clone https://github.com/your-username/python-setup-installer.git
   cd python-setup-installer
````

2. **(Optional but Recommended) Create a Virtual Environment**

   ```bash
   python -m venv env
   env\\Scripts\\activate   # On Windows
   source env/bin/activate  # On macOS/Linux
   ```

3. **Run the Installer Script**

   ```bash
   python install_semi_full_packages.py
   ```

4. Sit back and relax üòé. The script will install all required libraries automatically.
